Welcome to HTML!

HTML stands for Hypertext Markup Language, and is the foundational language of all websites! Since HTML is a markup language, it's not a programming language in the traditional sense. HTML's purpose is to organize and provide the bare-bones structure to web content. Think of it like a skeleton; a skeleton organizes organs into various areas of your body, and it also provides a structure for those organs to attach to and work from. It keeps stuff like eyes, ears, and your brain in the head; the heart, stomache, and lungs in the body; and toes and muscles in the foot.

Although you could make a fully functioning website purely with HTML, just like how you wouldn't consider a skeleton a complete and finished person, we don't typically see websites made with only HTML. We use other tools to build upon the structure that HTML gives us, such as CSS and JavaScript. However, since this tutorial is focused on giving you an introductory understanding of HTML itself, we won't be diving into those other tools just yet (although keep a lookout for new programming language tutorials!).


First, let's get started with the software you'll need for this tutorial.

What You'll Need

  • A web browser
  • An IDE

And that's it! While the second one may sound intimidating, don't worry; once you get started working with it, it'll all be pretty straightforward.

Let's take a look at each of these items.

Web Browser

Believe it or not, a web browser's job isn't just for surfing the web; it's used to develop websites as well!

This helps unveil what a web browser truly does; a web browser receives a website's code and translates it into the website that you end up seeing on your screen. This is the difference between a web browser and a search engine; a search engine fetches websites relevant to what you typed in the searchbar, while a web browser takes the code from the website that the search engine found and puts it in a format that you can see.

IDE

IDE stands for "Integrated Development Environment," which might sound scary and intimidating, but it's just something that can run and edit code, all bundled into one handy piece of software!


I will be using Firefox and Visual Studio Code as my web browser and IDE for this tutorial, however most web developers use a combination of Google Chrome and Visual Studio Code; although any web browser and HTML IDE will work just fine.

You can download Visual Studio Code here.



Creating Your First HTML File

Before you open your fancy new IDE, we first need to establish a directory where we will keep our HTML files. A directory is just a fancy way of saying 'folder' and is just a place to store data such as text files or images.

Go ahead and open File Explorer (or Finder, if you're using a Mac) and choose a place you'd like to store your HTML file(s).


Now remember where you put this folder, as we will need to find it later when we open up our IDE.

A Body of Tags

In our analogy, if HTML is the skeleton, its tags are the bones. Tags are the building blocks of HTML, each serving a specific purpose. For instance, a <p> tag represents a paragraph, while an <h1> tag is a heading. These tags work together to create a cohesive structure for your web content. 



    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Webpage!</h1>
        <p>This is a paragraph of text on my webpage.</p>
    </body>
    </html>


Anatomy of a Webpage

Let's break down the HTML structure in the above example.


  • <!DOCTYPE html>: This declaration tells the browser that we're using HTML5, the latest version of HTML.
  • <html>: The root element, encapsulating the entire webpage.
  • <head>: Contains meta-information about the document, such as the title.
  • <title>: Sets the title displayed in the browser tab.
  • <body>: Houses the content of your webpage.
  • <h1> and <p>: Headings and paragraphs, examples of HTML elements.


More Than Just HTML?

Sure, you could make a simple website using only HTML, but it's like having a house with just walls—functional but lacking style. We enhance our websites with CSS (Cascading Style Sheets) for design and JavaScript for interactivity. They're the interior decorators and electricians of our digital space.



Congratulations!

You've taken your first steps into the captivating world of HTML! You now understand the essential structure of a webpage, from the doctype declaration to the body content. Remember, HTML is the foundation upon which the digital realm is built, and your journey has just begun. As you continue to explore and experiment, keep an eye out for the next chapters where we'll dive deeper into the art of crafting stylish designs with CSS and adding interactivity with JavaScript. Happy coding, and may your web adventures be both exciting and rewarding!
